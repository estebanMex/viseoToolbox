Viseo Toolbox
=========

Simple toolbox for viseo teams 
### FEATURES 
- Example of use `generatorsnippets` npm module to generate snippets for vscode & sublimeText


## Installation
```shell
  git clone https://github.com/estebanMex/viseoToolbox.git
  cd viseoToolbox
  # install dependecies
  npm install
```

## Usage

```shell
    npm run gSnippets
```

## Tests

```shell
   npm test
```
